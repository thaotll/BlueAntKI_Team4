# BlueAntKI_Team4

Portfolio Analyzer mit KI-Integration.

## Setup

1. Python Virtual Environment erstellen:
```bash
cd backend
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
```

2. `.env` Datei erstellen (siehe `.env.example`)

3. Backend starten:
```bash
python run.py
```
